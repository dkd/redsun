Introduction

Redsun is a Redmine plugin that replaces the existing search with a fast Apache Solr powered engine.

Features

Redsun will respect any viewing restrictions on records to make sure people will only see results they are supposed to see.

Models

Currently only four models are being indexed. We plan to add news and attachments in the future.

  • Issues
  • Journals (=Comments)
  • Projects
  • WikiPages

Facets

  • Project
  • Author
  • Assigned To
  • Status
  • Tracker
  • Priority
  • Created on ** last 7 days ** last 30 days ** last 3 months ** last 6 months ** over 1 year ago
  • Updated on ** last 7 days ** last 30 days ** last 3 months ** last 6 months ** over 1 year ago #### Supported Redmine Versions

We have successfully used Redsun with Redmine 2.3-2.6. Please bear in mind that Redmine 3.x ist currently not supported, but we will probably create a compatible version soon.

Basic Installation

We recommend to test this plugin locally.

  • cd into your redmine root folder
  • cd plugins
  • git clone git@github.com:dkd/redsun.git redmine_redsun
  • run bundle install
  • rails g sunspot_rails:install
