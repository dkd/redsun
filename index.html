Introduction

Redsun is a Redmine plugin that replaces the existing search with a fast Apache Solr powered engine.

Supported Redmine Versions

We have successfully used Redsun with Redmine 2.3-2.6. Please bear in mind that Redmine 3.x ist currently not supported, but we will probably create a compatible version soon.

Basic Installation

We recommend to test this plugin locally.

  • cd into your redmine root folder
  • cd plugins
  • git clone git@github.com:dkd/redsun.git redmine_redsun
  • run bundle install
  • rails g sunspot_rails:install
